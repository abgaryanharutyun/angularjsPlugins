#hideOnBlure.js

#version 0.1 v
  Directive hideOnBlur for toggle open and Close
  
#Get Started
  
    <script src="../directives/hideOnBlur.js"></script>
  
#Usage
------------------

    <div class="clickedClass">
      <div hide-on-blure="clickedClass hide" >
      </div>
    </div>
------------------
#Class hide
------------------

    .hide{
    display:none;
    }
